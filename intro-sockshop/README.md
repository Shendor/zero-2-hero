---

## Demo

```bash
git clone https://github.com/microservices-demo/microservices-demo.git
cd microservices-demo/deploy/docker-only
export MYSQL_ROOT_PASSWORD=pw #windows -> 'set MYSQL_ROOT_PASSWORD pw'
docker-compose up -d
```
